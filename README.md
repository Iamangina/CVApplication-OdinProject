# CV Application - React
Demo: https://iamangina.github.io/CVApplication-OdinProject/

## Overview

This is a **React-based CV/Resume Builder** that allows users to create a professional CV, customize its style, and download it as a PDF. The application is interactive and supports personal information, education, work experience, and skills management.  

It uses **Vite** as a build tool and **html2pdf.js** for PDF generation. The application can be deployed on **GitHub Pages** for live demos.

---

## Features

- Add, edit, and remove **personal details** (name, photo, contact info, summary, desired position).  
- Add multiple **education** entries.  
- Add multiple **work experience** entries.  
- Add multiple **skills**.  
- **Delete individual entries** easily.  
- **Customize CV style**: font and color.  
- **Download CV as PDF** with proper formatting.
